Muse

Muse is an online repository for users to curate their collection music sets or performances from Youtube, Mixcloud, or Soundcloud. Written in Ruby on Rails, this was made during my 6th week of WDI.

  • Technologies used: Ruby
  • Ruby on Rails
  • Bootstrap CSS

  • Github Repo: Muse

Conshareto

Conshareto was a MEN-stack application that I built with a team of 5 during our 9th week of WDI. Conshareto was designed to be an easy-to-use, concert-managing application. Users can search for available concerts within a nearby radius using the Jambase API and track it if they are interested in that concert.

  • Technology used: Node.js
  • Express
  • MongoDB
  • Jamebase API
  • Bootstrap-CSS

  • Github Repo: Conshareto
  • Deployed Link: Conshareto

Coordinet

Coordinet is a location-sharing based chat application that allows users to share their location along with a graphical icon and a short message. Users can see whats going on around them and 'coordinet' with each other.

  • Technologies used: Node.js
  • Express
  • Angular.js
  • Sockets.io
  • Ionic Framework
  • Google Maps API

  • Github Repo: Coordinet
  • Deployed Link: Coordinet-Beta (ionic version will be uploaded to PlayStore soon!)
